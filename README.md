# sentence to parts
A Python module to decompose a sentence into parts

# Quick start
```
$ git clone https://github.com/Himmelstein/snt2parts
$ cd snt2parts
snt2parts $ make init
```

# Run test
```
snt2parts $ make test
```

# Generate documentation
```
snt2parts $ ./make_docu
```
Documentation is located at docs/html/
